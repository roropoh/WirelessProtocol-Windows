Running this program:
1. Connect the wireless modem
2. Run the executable ("Wireless-GRVM"), and press the Config tab
3. Go into the File tab, Send File

Psedo code, design, and test documents from PDF are all located in the "Documentation" folder.

All the source codes are in the "Wireless-GRVM Source Files" folder.
Project will be in the "Wireless-GRVM PROJECT FOLDER".
